Scala Puppet Module
====================

[![Build Status](https://secure.travis-ci.org/smarchive/puppet-scala.png)](http://travis-ci.org/smarchive/puppet-scala)

Overview
--------

Install the Scala programming language from the official project site.

Usage
-----

Example:

    class { 'scala':
      version => '2.9.2',
    }
